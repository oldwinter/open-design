参考商品工作流

一张商品照片，生成一组紧凑的电商图片。

V1 会把商品锁定在真实参考图上，然后生成一张主图、一张卖点图和一张生活方式图。

Mode reference-product
Scope 一个商品
Outputs 3 张图片 + manifest
已上传参考图 身份已锁定
  • 保留形状、颜色、材质、结构和比例。
  • 不要编造卖点、证书或测量值。
  • 最终 marketplace 合规和文字叠加仍需人工 review。

Main image

白色或柔和中性背景上的干净 packshot。

Feature image

一个已确认卖点，并为设计师后续添加标签预留空间。

Lifestyle image

可信的使用场景，同时保持商品忠于参考图。
